 This project is parses a file by identifiers, real, and integers.
 <Identifier>
  A sequence of letters or digits, however the first character must be a letter and last char must be either $ or a letter.
  
 <Integer>
  An unsigned decimal integer i.e., a sequence of decimal digits.

  <Real> 
    integer followed by “.” and Integer, e.g., 123.00

  <Keywords>
Some identifiers are reserved for use as keywords, and may not be used otherwise:
       e.g.,  int, if, else, endif,  while, return, get, put   etc.
